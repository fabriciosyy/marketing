PUBLICAÇÃO NO GITHUB
1. Envie index.html, portfolio.html e a pasta assets para a raiz do repositório.
2. Em Settings > Pages, escolha Deploy from a branch.
3. Selecione main e /(root).

COMO TROCAR IMAGENS DO PORTFÓLIO
1. Salve a imagem em assets, por exemplo: assets/projeto-01.jpg
2. Em portfolio.html, substitua:
   <div class="media"></div>
   por:
   <div class="media"><img src="assets/projeto-01.jpg" alt="Nome do projeto" style="width:100%;height:100%;object-fit:cover"></div>
